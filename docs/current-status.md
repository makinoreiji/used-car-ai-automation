# PJ Current Status

このファイルを見れば、現時点のPJ状況、課題、方針、進行中の作業が分かる状態にする。

## PJの目的

中古車業務における画像処理・書類読み取り・アノテーション・モデル改善の流れを整備し、AIを継続的に改善できる状態を作る。

## 現在の主な論点

### 1. 要件・TOBEがまだ固まっていない

渡辺さん側でTOBEを文章化する宿題が残っている。  
CAWAL側としては、TOBEが出てきた時に議論できるよう、現行フロー・課題・技術選定の判断材料を整理している。

### 2. AIを継続改善できる標準ワークフローが整っていない

標準ワークフローの叩き台と、各工程で「やりたいこと」までは作成済み。  
ただし、アノテーションや教師データの形式はモデル自体の処理フローに依存するため、現時点ではモデル処理に依存しない粒度で整理している。

### 3. モデル自体の処理フローに改善の余地がある

現行フロー:

```text
分類（内装・外装・広告）
  -> 検知（ナンバー・バナー）
  -> セグメンテーション
  -> インペイント
```

改善案:

```text
詳細分類（その画像がどういう画像なのかを認識）
  -> ルールベースで後続処理を振り分ける
```

詳細分類はまだ採用決定ではない。  
まずは分類モデルの候補としてResNet50を中心に学習・整理し、萩原さんと議論できる状態を作る。

### 4. 個人の画像認識知見が不足している

モデル自体の処理フローについて具体的に議論するため、画像分類、検知、セグメンテーション、インペイントの基礎と事例を学習中。
